# Start by Task

Pick a task to jump into the right guides.

- Start a coding project → environments, structure, and version control
- Run an fMRI analysis → setup, preprocessing, and first-level models
- Set up a behavioural experiment → online (jsPsych/PsychoPy) and on-site
- Request ethics approval → MEC/SMEC routes and checklists
